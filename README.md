# React Ecommerce Project
🛒 React E-Commerce Application

📌 Project Overview

This project is a React-based E-Commerce Web Application built using modern frontend technologies.
It allows users to browse products, search items, and manage a shopping cart with a clean and responsive UI.

---

🎯 Problem Statement

To build a scalable and user-friendly e-commerce interface that demonstrates:

- Component-based architecture
- State management
- Real-time UI updates

---

🛠️ Tech Stack

- Frontend: React (Vite), JavaScript (ES6+)
- State Management: Context API
- Routing: React Router
- Styling: CSS
- Build Tool: Vite

---

✨ Features

- 🔍 Product Search (real-time filtering)
- 🛒 Add to Cart functionality
- 📦 Dynamic product listing
- ⚡ Fast UI rendering using React Hooks
- 🎨 Clean and responsive UI

🚀 Installation & Setup

1️⃣ Clone Repository

cd react-ecommerce-pro

2️⃣ Install Dependencies

npm install
